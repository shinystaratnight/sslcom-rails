Lotus Domino Server versions 4.6x and 5.0x

Creating a key ring and a request for an Internet server certificate
Follow the steps below to create a key ring file and a certificate request that can be sent to VeriSign.
    For R5x, begin here and complete .
  1. Launch the Domino Administration client.
  2. Select File-Open Server and select the Domino server you wish to administer
  3. Click the file tab
  4. Double click on Server Certifcate Administration database (certsrv.nsf)
  5. From the administration panel, click System Databases and choose Open Domino Server Certificate Administration (CERTSRV.NSF) on the local machine.
  6. Click Create Key Ring.
  7. Enter a name for the key ring file in the "Key Ring File Name" field.
  8. Enter a password for the server key ring file in the "Key Ring Password" field.

    Note Password is an alphanumeric set of characters that protects the key ring from unauthorized use. The password is case sensitive. You should specify at least 12 alphanumeric characters for the password.
  9. Select a key size. This is the size Domino uses when creating the public and private key pairs. The larger the size, the stronger the encryption. If you are using the international version of Domino, only the 512 bit key size will work for you unless you have Release R5.04.
  10. Specify the components of your server's distinguished name.
  11. Click Create Key Ring.
  12. After you read the information about the key ring file and distinguished server name, click OK.
  13. Click Create Certificate Request.
  14. Do one of the following:

    (Recommended) Click "Paste into form on VeriSign's site."

    Note If you are submitting a request to VeriSign, use this method. VeriSign does not use PKCS format for requests sent by e-mail. Click "Send to CA by e-mail" and enter the VeriSign's e-mail address and information about your e-mail address, phone number, and location.
  15. If you want to log information about this request in the Server Certificate Administration application, select Yes in the "Log Certificate Request" field. Otherwise, select No.
  16. Click Create Certificate Request.
  17. Enter the password of the key ring file that you specified in step 4.
  18. If you selected "Paste into form on VeriSign's site" in step 10, highlight the text in the dialog box , press CTRL+C to copy it to the Clipboard, and click OK. Using a browser, visit the VeriSign site and follow the instructions provided on submitting a request for a new certificate.


Note You must select all the text in the second dialog box, including Begin Certificate and End Certificate.

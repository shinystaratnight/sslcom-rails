Generate a certificate request file, PKCS10
  1. Click VPN, and go to the Certificates Tab
  2. Click the Certificate Request link
  3. Fill out the certificate request form. The IP address will be the interface you want to manage via ssl. The certificate will specifically be used for that particular IP address. In other words, you can only use one certificate per interface.
  4. Click Generate and it will create a pkcs10 file, which will contain a hash of the above information, and will be interpreted by the CA, which in our example would be Microsoft.

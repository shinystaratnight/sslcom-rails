This document introduces generating a Key Pair and CSR (Certificate Signing Request), and answers questions you might have.

Generating a Key Pair and CSR for O'Reilly WebSite Professional Version 2.x and Higher
WARNING: BACK UP YOUR WEBSITE.KEY FILE. This is a crucial safety precaution in case anything goes wrong. The key file is located at C:\Website\Admin

Note: This information applies to WebSite Professional 2.x and higher.
  1. Open WebSite Professional's Server Properties. Enter the key ring password when prompted.
  2. Display the Key Ring tab.
  3. Click the New Key Pair button and complete the steps in the wizard. At the end of the process you will have a new key pair and CSR.
  4. Back up your key file (website.key) and CSR on a floppy disk and store the disk in a secure location. If you lose your private key or forget the password, you will not be able to install your Secure Server ID and will need to request and purchase a new one from VeriSign.
  5. Go back to the Verisign enrollment pages and follow the instructions in Enrollment Step 4 to submit your CSR.


Renewing Your Secure Server ID for WebSite Professional 2.x and Higher
You must renew your Secure Server ID annually. Thirty days before your Secure Server ID's expiration date, VeriSign sends a notification to your technical and organizational contacts. Secure Server IDs can only be renewed during the 30 days prior to expiration. During this time, both your new and old certificates are active, allowing a smooth transition to the new certificate without any interruption in service.

To renew your Secure Server ID:
  1. Back up your current certificate database ( C:\website\admin\website.key) to a floppy disk.
  2. Open Server Properties and display the Key Ring tab.
  3. Right click on the certificate key to be renewed. If your server has multiple identities, you may have more than one certificate. Be sure to choose the correct certificate to renew.
  4. From the context menu, select Certificate Renewal Request.
  5. Provide an appropriate location to save the new request.
  6. Go back to the Verisign renewal pages to submit your renewal request. Your renewed certificate will be emailed to you.
  7. Save the renewed certifcate from your email to a file.
  8. Open Server Properties and display the Key Ring tab.
  9. Right click on the certificate key you are renewing and select Attach Certificate to Key Pair.
  10. Locate the file in which you saved the renewed certifcate and click Open.
  11. Once the renewed certificate is attached, close Server Properties and back up your updated website.key file.

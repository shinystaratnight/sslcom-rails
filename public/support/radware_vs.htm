Generating a certificate request
To generate a certificate signing request (CSR), use the following command:

Certificate Req
This command creates the CSR for a specific key, to be sent to VeriSign.

Syntax:
certificate req <keyID>

Variable Description
keyID The unique identification of the key.

A certificate request for the key with an ID of 7 is created.

To export the certificate request, use the following command:

Certificate Export
This command exports a certificate with a specific keyID. Syntax:

certificate export <keyID>

Variable Description
keyID The unique identification of the key.


Example:
certificate export 4

This exports the certificate associated with keyID 4. You will be prompted to choose between two export protocols:

1. Zmodem: Sent to the terminal.
2. ASCII: Printed to screen for cut and paste.

Installing a Certificate
After receiving the certificate from VeriSign, install it via the following command:

Certificate Import
This command imports a certificate and associates it with a keyID.

Syntax:
certificate import <keyID>

Variable Description
keyID The unique identification of the key.


Example:
certificate import 4

This imports a certificate and associates it with keyID 4. You will be prompted to choose between two import protocols:

1. Zmodem: From the terminal.
2. ASCII (Cut & Paste): Cut and then paste into the HyperTerminal Application.

Note:
In ASCII, the certificate is pasted into the appropriate field when prompted.
